## 一、项目原创内容

以下内容属于本项目围绕高职涉农物联网专业校本教研、岭南红橙病虫害识别与智能防治场景形成的原创或二次开发成果：

1. 项目应用场景设计：面向岭南红橙病虫害智能监测、实训教学、校本教研、检测台账和教学案例生成的一体化业务流程设计。
2. 桌面应用工程代码：`app.py`、`lingnan/`、`tools/`、`tests/` 中围绕本项目业务编写的 Python 代码、PyQt5 界面逻辑、检测流程、台账管理、报表导出、教研生成、测试用例和文档生成脚本。
3. 校本知识库组织方式：围绕 12 类红橙/柑橘高发病虫害、5 个物候期、三位一体绿色防治方案、安全间隔期（PHI）提示所构建的本地 SQLite 结构、字段组织和检索逻辑。
4. 教学教研生成流程：将 YOLOv8 检测结果、严重程度、物候期、农户/果园档案、绿色处方和可选大模型接口转化为教学案例、学生农技实训指导意见的 Prompt 模板、降级模板和审核导出流程。
5. 项目交付文档：安装手册、项目开发记录、项目使用手册、开发与应用报告等由本项目上下文生成和整理的交付性文档。

# 一、案例概述｜2分钟以内

## 第1页：案例背景与应用场景｜约 40 秒

**PPT内容**：

- 面向高职涉农物联网专业实训教学
- 依托广东省特色创新类项目（2025KQNCX312）
- 聚焦岭南红橙病虫害识别、防治指导与校本教研
- 应用场景：课堂案例教学、实训室检测、田间图片分析、教研资料生成

**录屏/画面建议**：

- 展示系统首页或项目标题页
- 可穿插红橙病虫害图片、实训课堂或系统主界面截图

**解说要点**：

本案例来源于真实农业生产与高职涉农物联网专业教学需求，将省特色创新项目中的红橙病虫害图像数据和物联网环境数据转化为可直接用于课堂和实训的数字化教学资源。

## 第2页：解决的主要问题｜约 50 秒

**PPT内容**：

- 科研成果转化慢，课题成果难以及时进入课堂
- 实训教学缺乏真实复杂农业生产情境
- 学生学习过程、识别能力和实训成果评价维度单一
- 教师备课、案例整理、台账归档耗时高

**录屏/画面建议**：

- 使用“传统教学痛点 vs 智能系统支撑”的对比图
- 展示传统图片识别教学、纸质记录与系统化台账的对比

**解说要点**：

传统课堂往往停留在看图识病和经验讲解，学生难以接触真实复杂农情数据。系统通过 AI 检测、知识库推理和教研生成，帮助教师把科研成果快速转化为实训案例。

## 第3页：总体解决思路｜约 30 秒

**PPT内容**：

流程闭环：

1. 真实红橙病虫害图像
2. 物联网温湿度等环境数据
3. YOLOv8s-XH 本地识别
4. SQLite 校本知识库推理
5. 三位一体防治方案
6. 教学案例与实训指导生成
7. 台账归档与评价

**录屏/画面建议**：

- 展示系统功能导航
- 使用流程图突出“监测—识别—决策—教研—评价”闭环

**解说要点**：

平台不是单一识图工具，而是将智能监测、农技决策、教学案例生成和实训评价连接起来，形成面向教学应用的完整闭环。

# 二、实现功能｜5分钟以内

## 第4页：系统总体架构｜约 40 秒

**PPT内容**：

- 前端交互：PyQt5 桌面端，大字体、高对比度、单键触发
- 视觉识别：YOLOv8s-XH 本地离线推理
- 数据存储：SQLite 校本知识库与历史台账
- 报表导出：Excel / PDF / Markdown
- 教研生成：模板生成 + 可选大模型增强

**录屏/画面建议**：

- 展示系统主界面和左侧功能导航
- 快速切换“智能检测”“教学生成”“历史台账”等页面

**解说要点**：

系统采用本地化部署架构，核心功能断网可用。前端负责交互和可视化，算法层完成病虫害识别，数据层保存知识库和台账，教研层将检测结果转化为教学材料。

## 第5页：功能一：病虫害智能识别｜约 60 秒

**PPT内容**：

- 支持图片、视频、摄像头检测
- 秒级识别 12 类高发病虫害
- 输出检测框、类别、置信度
- 支持红蜘蛛、蚜虫、木虱等群聚虫害计数
- 自动判断轻度、中度、重度

**录屏/画面建议**：

1. 打开“智能检测”页面
2. 导入一张红橙病虫害图片
3. 点击开始检测
4. 展示检测框、病虫害名称、置信度和严重程度

**解说要点**：

系统本地部署 YOLOv8s-XH 模型，可在无网络环境下完成病虫害识别。对于红蜘蛛等小目标和群聚虫害，系统还能进行计数和严重程度评估。

## 第6页：功能二：绿色防治知识库推理｜约 50 秒

**PPT内容**：

- 联动病虫害类别、物候期、温湿度等数据
- 匹配 SQLite 校本知识库
- 生成物理、生物、合规化学“三位一体”方案
- 突出安全间隔期 PHI
- 黄龙病等高危病害给出强提醒

**录屏/画面建议**：

- 展示检测结果右侧的防治方案
- 切换不同物候期，展示防治建议变化

**解说要点**：

系统不是只给出识别结果，还会结合物候期和知识库推送精准防治建议。化学防治部分突出安全间隔期，帮助学生理解绿色防控和规范用药要求。

## 第7页：功能三：农情台账与报表导出｜约 45 秒

**PPT内容**：

- 自动保存检测记录
- 支持农户、果园、病虫害、时间筛选
- 3 秒内离线生成 Excel 教学台账
- 支持 PDF 分析报告
- 服务课堂评价与教研归档

**录屏/画面建议**：

1. 进入“历史台账”页面
2. 按时间或病虫害类别筛选
3. 点击导出 Excel
4. 展示导出的台账文件

**解说要点**：

检测结果自动沉淀为农情台账，教师可以将其作为学生实训过程记录、课堂评价依据和教研归档材料，减少手工整理成本。

## 第8页：功能四：校本教研内容生成｜约 60 秒

**PPT内容**：

- 生成《岭南红橙实时防害教学案例》
- 生成《学生农技实训指导意见》
- 支持教师审核、修改、导出
- 将检测结果转化为课堂案例、实训任务和考核材料
- 可选接入 DeepSeek、豆包或本地大模型

**录屏/画面建议**：

1. 进入“教学生成”页面
2. 选择文档类型
3. 点击生成草稿
4. 展示教学案例或实训指导意见
5. 勾选审核并导出

**解说要点**：

系统可以把一次真实检测结果转化为教学案例和学生实训指导。教师可审核、修改后导出，实现从生产数据到教研资源的快速转化。

## 第9页：应用过程演示闭环｜约 45 秒

**PPT内容**：

完整应用流程：

1. 导入红橙病虫害图片
2. YOLOv8s-XH 本地识别
3. 选择物候期/读取环境数据
4. 知识库生成防治方案
5. 保存检测台账
6. 导出报表
7. 生成教学案例

**录屏/画面建议**：

- 用连续录屏快速串联完整操作
- 每一步保留 3 至 5 秒画面

**解说要点**：

通过这套流程，学生可以从真实图片出发，完成识别、分析、防治和记录；教师则可以同步获得课堂案例和实训评价材料。
